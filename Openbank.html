Developer Console

Open your browser's developer console (F12) to interact with the data directly: 

// Update account balance
await OpenBankData.updateAccountBalance('acc_main', 3500);

// Add transaction
await OpenBankData.addTransaction({
    accountId: 'acc_main', 
    amount: -75, 
    description: 'Coffee Shop'
});

// Toggle card status
await OpenBankUtils.toggleCard('card_001');

// Update settings
await ConfigManager.updateConfigProperty('settings', 
    'userPreferences.dashboard.theme.mode', 'dark');

// Export all data
ConfigManager.exportConfig('all');
